# Corne (Helidox)

El teclado Corne tiene un diseño dividido ergonómico en el que destacan columnas escalonadas de 3x6 con un clúster de 3 teclas para los pulgares. El layout del Corne tiene muchas caracteristicas distintivas que lo destacan de otros teclados ergonómicos. El layout pequeño deja todas las teclas a solo una unidad de la fila principal o "home row". El teclado tambien incluye huellas para iluminación RGB debado de cada tecla asi como 6 LEDs por lado que apuntan hacía la mesa. La PCB cuenta también con huellas para pantallas OLED. Por defecto, las pantallas muentras la capa actual y la información de las teclas presionadas recientemente.

## Conexión del teclado

Para comenzar a utilizar tu teclado Corne, es necesario que conozcas algunos detalles importantes que evitarán daños en tu teclado:

1. El cable USB debes conectarlo al lado **IZQUIERDO** ya que este actua como Maestro (master). El lado derecho actua como Esclavo (slave).
2. Conecta primero el cable TRRS en ambas partes (izquiera y derecha), luego conecta el cable USB. Nunca conectes o desconectes el cable TRRS mientras el cable USB esté conectado ya que podrias generar un corto y dañar los microcontroladores.

## Configuración

Tu teclado Corne viene con el firmware QMK preinstalado y con la compatibilidad de VIA habilitada, lo cual te permite de manera fácil y rápida configurar el teclado a tu gusto. Para comenzar a utilizar el teclado no requieres hacer ninguna configuración ya que viene con la predeterminada de Corne.

Puedes ajustar la posición o función de cada tecla, crear macros, configurar las capas, configurar la iluminación RGB, importar y exportar tus configuraciones.

Sigue los pasos de la Guía de configuración con VIA para personalizar tu teclado.
